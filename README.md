# 🪟 Glassmorphism Login Page



Projeto de uma tela de login moderna com efeito de vidro (Glassmorphism), desenvolvida durante meus estudos de Front-End. O foco principal foi a criação de uma interface limpa, responsiva e otimizada para dispositivos móveis.

---

## 🚀 Tecnologias

As seguintes ferramentas foram usadas na construção do projeto:

- [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML)
- [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) (Flexbox, Media Queries, Backdrop Filter)
- [Font Awesome](https://fontawesome.com/) (Ícones)

## 📱 Responsividade

O projeto foi ajustado para garantir uma boa experiência em qualquer dispositivo:
- **Desktop:** Layout centralizado e fixo.
- **Mobile:** Ajuste automático da largura (90%), fontes otimizadas para evitar zoom automático no iOS/Android e bloqueio de rolagem lateral.

## 🎨 Demonstração

O layout utiliza `backdrop-filter: blur(10px)` para criar o efeito de desfoque no fundo, mantendo a legibilidade dos campos de entrada de e-mail e senha.
